IOTA-TSC

  • Task Schedule Center
  • 对结构物dimension调度
  • scheme决定调度时机，dimCap表示调度对象
Features
  • dimension --> schedule转换器(Repeat模式)
  • event处理：start,pause,resume,shutdown
  • 并发调度和串行调度
  • DeviceInterface并发能力控制
  • 监听Thing更新
  • 监听Driver处理阶段变化(监测/恢复用)
  • 日志写入(未做)
  • 硬关闭后的恢复能力(未做完)

Example

    import "iota/scheduler/thing"
    s := thing.NewThingScheduleService(thingGetter ThingsGetter, dcDriver DimCapDriver)
    s.Start()
    s.Pause()
    s.Resume()
    s.Shutdown()
    s.GetSchedulers()

增强cron

  • 基于robfig/cron , a cron library for go
  • 支持@now == @every 1s,立即执行
  • 支持@every {n}M,月份的interval支持
